Mendel

Mendel Programming Language

Mendel is a statically typed, multi-paradigm, and compiled programming language with modern and clean style. The Mendel compiler was developed in JavaScript (and will migrate to Mendel itself in the future). Mendel can be compiled to low level subsets of JavaScript and C.

Mendel is a project created in HackOHI/O. For more information about the project and out team, visit our presentation.

Key features of Mendel Programming Language:

  • Template-based generic
  • Prototype-based OOP
  • Multi-paradigm: Proc + Func + OO
  • Static & strong typing
  • Full type inference
  • Clean, modern and flexible design
  • Compiled to JavaScript & C
  • Simple enough to be built in 24 hours

Design Purpose

Mendel language is a programming language designed and implemented by me and four of my friends. We decided to make this language a tool for students from CS major to non-CS major to learn programming. Therefore, the syntax of this programming language is very simple compared to most current languages, so that beginners can better focus on how to instruct computers to do work rather than trivial syntax.

Meanwhile, the essence of Mendel programming language is extremely easy. So we believe it could also be a example to educate compiler theory. Since it only took us 24 hours for five students to implemented it, it would also be simple enough for CS students to learn the idea behind this compiler and thereby learn the theory of compiling a high-level language.
