Statement of Career Goal

I career goal is to become a software engineer that is able to design and implement efficient computer systems that can be used by millions of people around the world. And my specialization is computer graphics and dealing with low-level graphics program like openGL and DirectX and GUI libraries. At the same time, I’m familiar many other programming fields, including front-end and back-end for websites, sever programming, and database programming.

I’ve worked as intern twice. One is the peer leader in a research team at the Chinese Sciences Academy, Institute of Automation. During the period, my focus was to work with peers to desgin an algorithm to recognize license plates from a given video. I worked with Matlab some usual approach like wave filtering and some pattern regonition. The other one is a web front-end position at iRobotMatrix, my job generally works with the company’s websites especially in implement responsive pages with Tachyons frameworks, as well as writing unit tests for company’s java libraries.

I myself also created some project during my free time. One significant is the compiler for Mendel programming language, which I implement with four of my friends using node.js . I was mostly responsible for writing the compiler’s code generation with general semantics and standard library. We were able to generate code to C and a JavaScript subset. The other one is a JSON parser using C++ which I did on my own. The purpose of this project was mainly practice C++11 syntax I just learned and arrange codes including library codes, program codes, and test codes in a project using Cmake build system.

As a mentioned in my resume, I am proficient with many programming languages and frameworks before I study at college. Meanwhile, I plan to do further study about software engineering during college study, including general software engineering approach, rationale about big topics in compute science like computer network, and deep special knowledge about computer graphics like advanced approach in 3D modeling and real-time rendering. In the near future, I will be able to use all my skills to write computer programs serving users all over the world.
